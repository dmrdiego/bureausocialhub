---
description: Sincronizar alterações com o GitHub (Commit e Push)
---
1. Adicionar todas as alterações:
   ```bash
   git add .
   ```
2. Realizar o commit (substitua a mensagem conforme necessário):
   ```bash
   git commit -m "Update: Melhorias e correções recentes"
   ```
3. Enviar para o repositório remoto:
   ```bash
   git push origin main
   ```
