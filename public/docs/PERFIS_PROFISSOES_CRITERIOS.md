# PERFIS DE PROFISSÕES E CRITÉRIOS DE SELEÇÃO

## Programa Moradia Artesãos

---

**Instituto Português de Negócios Sociais – Bureau Social**

---

## Introdução

Este documento estabelece os **perfis profissionais** elegíveis para o Programa Moradia Artesãos, bem como os **critérios de avaliação** específicos para cada categoria de ofício tradicional. O objetivo é garantir uma seleção justa, transparente e alinhada com os objetivos de preservação do património cultural português.

---

## PARTE I — CATEGORIAS DE OFÍCIOS TRADICIONAIS

### 1. Têxteis e Vestuário

#### 1.1 Alfaiate

| Aspeto | Descrição |
|--------|-----------|
| **Definição** | Profissional especializado na confeção de vestuário masculino por medida |
| **Técnicas essenciais** | Tirar medidas, corte de tecidos, costura à mão e máquina, provas e ajustes, acabamentos |
| **Produtos típicos** | Fatos, casacos, calças, coletes, camisas por medida |
| **Equipamento necessário** | Máquina de costura industrial, mesa de corte, ferro de engomar profissional, manequim |
| **Espaço recomendado** | 25-40 m² |
| **Nível de ruído** | Baixo |
| **Requisitos especiais** | Iluminação natural abundante |

**Critérios de avaliação específicos:**

| Critério | Indicadores |
|----------|-------------|
| Domínio técnico | Qualidade do corte, precisão das costuras, acabamentos |
| Portfolio | Mínimo 5 peças completas documentadas |
| Experiência | Preferência para formação tradicional (mestre-aprendiz) |

#### 1.2 Bordadeira

| Aspeto | Descrição |
|--------|-----------|
| **Definição** | Artesã especializada em técnicas de bordado manual |
| **Técnicas essenciais** | Ponto cruz, ponto cheio, ponto de Arraiolos, bordado de Castelo Branco, bordado de Viana |
| **Produtos típicos** | Toalhas, lençóis, almofadas, quadros, vestuário bordado |
| **Equipamento necessário** | Bastidores, agulhas, linhas, tecidos, padrões |
| **Espaço recomendado** | 15-25 m² |
| **Nível de ruído** | Muito baixo |
| **Requisitos especiais** | Iluminação de qualidade, ambiente calmo |

#### 1.3 Tecelão / Tecelã

| Aspeto | Descrição |
|--------|-----------|
| **Definição** | Artesão que opera teares manuais para produção de tecidos |
| **Técnicas essenciais** | Preparação de urdidura, tecelagem em tear manual, acabamentos |
| **Produtos típicos** | Mantas, tapetes, tecidos para vestuário, panos decorativos |
| **Equipamento necessário** | Tear manual (vertical ou horizontal), urdideira, lançadeiras |
| **Espaço recomendado** | 30-50 m² (para acomodar tear) |
| **Nível de ruído** | Médio |
| **Requisitos especiais** | Pé-direito adequado, piso resistente |

#### 1.4 Rendeira

| Aspeto | Descrição |
|--------|-----------|
| **Definição** | Artesã especializada na produção de rendas tradicionais |
| **Técnicas essenciais** | Rendas de bilros, rendas de agulha, rendas de crochet |
| **Produtos típicos** | Toalhas, guardanapos, aplicações para vestuário, peças decorativas |
| **Equipamento necessário** | Almofada de bilros, bilros, alfinetes, padrões |
| **Espaço recomendado** | 15-20 m² |
| **Nível de ruído** | Muito baixo |
| **Requisitos especiais** | Iluminação de qualidade |

## PARTE II — MATRIZ DE CRITÉRIOS DE SELEÇÃO

### Critérios Gerais (aplicáveis a todos os ofícios)

| Critério | Peso | Descrição | Pontuação |
|----------|------|-----------|-----------|
| **Domínio técnico** | 30% | Qualidade e diversidade do trabalho apresentado | 0-100 |
| **Situação socioeconómica** | 25% | Grau de carência habitacional e económica | 0-100 |
| **Transmissão de saberes** | 20% | Disponibilidade e capacidade para ensinar | 0-100 |
| **Projeto de atividade** | 15% | Viabilidade e qualidade do plano de trabalho | 0-100 |
| **Ligação à comunidade** | 10% | Integração local e contributo esperado | 0-100 |

### Tabela de Pontuação — Domínio Técnico (30%)

| Nível | Pontuação | Descrição |
|-------|-----------|-----------|
| **Mestre** | 90-100 | Domínio completo do ofício, capacidade de inovação, reconhecimento público |
| **Oficial** | 70-89 | Domínio sólido das técnicas principais, trabalho de qualidade consistente |
| **Aprendiz avançado** | 50-69 | Conhecimento das técnicas básicas, necessita aperfeiçoamento |
| **Aprendiz** | 30-49 | Em fase de aprendizagem, potencial demonstrado |
| **Iniciante** | 0-29 | Conhecimento muito limitado, não elegível |

### Tabela de Pontuação — Situação Socioeconómica (25%)

| Nível | Pontuação | Descrição |
|-------|-----------|-----------|
| **Carência grave** | 90-100 | Sem habitação ou em situação de risco iminente |
| **Carência elevada** | 70-89 | Habitação inadequada ou renda superior a 50% do rendimento |
| **Carência moderada** | 50-69 | Renda entre 35-50% do rendimento, condições razoáveis |
| **Carência ligeira** | 30-49 | Renda até 35% do rendimento, condições adequadas |
| **Sem carência** | 0-29 | Situação habitacional estável, não prioritário |

### Tabela de Pontuação — Transmissão de Saberes (20%)

| Nível | Pontuação | Descrição |
|-------|-----------|-----------|
| **Muito disponível** | 90-100 | Experiência de ensino, disponibilidade total, metodologia definida |
| **Disponível** | 70-89 | Disponibilidade para ensinar, alguma experiência |
| **Moderadamente disponível** | 50-69 | Disponibilidade limitada, sem experiência de ensino |
| **Pouco disponível** | 30-49 | Disponibilidade mínima, foco no trabalho individual |
| **Indisponível** | 0-29 | Sem disponibilidade para transmissão de saberes |

### Tabela de Pontuação — Projeto de Atividade (15%)

| Nível | Pontuação | Descrição |
|-------|-----------|-----------|
| **Excelente** | 90-100 | Plano detalhado, viável, inovador, com mercado identificado |
| **Bom** | 70-89 | Plano consistente, viável, com potencial de mercado |
| **Satisfatório** | 50-69 | Plano básico, necessita desenvolvimento |
| **Insuficiente** | 30-49 | Plano vago, viabilidade questionável |
| **Inexistente** | 0-29 | Sem plano de atividade definido |

### Tabela de Pontuação — Ligação à Comunidade (10%)

| Nível | Pontuação | Descrição |
|-------|-----------|-----------|
| **Forte** | 90-100 | Residente na zona, participação ativa em associações locais |
| **Moderada** | 70-89 | Ligação à zona, conhecimento da comunidade |
| **Ligeira** | 50-69 | Sem ligação prévia, mas com interesse demonstrado |
| **Fraca** | 30-49 | Sem ligação, interesse limitado |
| **Inexistente** | 0-29 | Sem qualquer ligação ou interesse |

## PARTE III — PROCESSO DE AVALIAÇÃO

### Etapa 1: Análise Documental

A Comissão de Seleção analisa a candidatura e documentos anexos, verificando:

| Elemento | Verificação |
|----------|-------------|
| Formulário | Preenchimento completo e correto |
| Documentos de identificação | Validade e autenticidade |
| Comprovativos de rendimentos | Conformidade com declarações |
| Portfolio | Qualidade e autenticidade do trabalho |
| Certificados | Validade e relevância |

### Etapa 2: Entrevista

Os candidatos pré-selecionados são convocados para entrevista, onde se avalia:

| Aspeto | Objetivo |
|--------|----------|
| Percurso profissional | Compreender a trajetória no ofício |
| Motivação | Avaliar o compromisso com o Programa |
| Projeto de atividade | Detalhar o plano de trabalho |
| Disponibilidade | Confirmar disponibilidade para obrigações |
| Expectativas | Alinhar expectativas com a realidade do Programa |

### Etapa 3: Visita ao Local de Trabalho (se aplicável)

Quando o candidato já exerce o ofício, pode ser realizada visita para:

| Objetivo | Descrição |
|----------|-----------|
| Verificar condições de trabalho | Avaliar o ambiente e equipamentos atuais |
| Observar o trabalho | Ver o artesão em ação |
| Conhecer produtos | Avaliar a qualidade e diversidade da produção |
| Conversar com clientes | Obter referências (se possível) |

### Etapa 4: Deliberação

A Comissão de Seleção reúne para:

1. Atribuir pontuação a cada critério
2. Calcular pontuação final ponderada
3. Ordenar candidatos por pontuação
4. Selecionar beneficiários (conforme vagas disponíveis)
5. Elaborar lista de espera
