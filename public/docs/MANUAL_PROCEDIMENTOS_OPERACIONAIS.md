## 1. Introdução

### 1.1 Objetivo do Manual

Este Manual de Procedimentos Operacionais estabelece os processos, fluxos de trabalho e responsabilidades para a gestão do **Programa Moradia Artesãos** do Instituto Português de Negócios Sociais – Bureau Social.

O Manual destina-se a orientar a equipa técnica do Instituto na execução das suas funções, garantindo a uniformidade, eficiência e qualidade na prestação de serviços aos beneficiários.

### 1.2 Âmbito de Aplicação

O presente Manual aplica-se a todos os colaboradores do Instituto envolvidos na gestão do Programa Moradia Artesãos, incluindo:

- Direção
- Técnicos de serviço social
- Técnicos administrativos
- Colaboradores externos e voluntários

### 1.3 Revisão e Atualização

O Manual é revisto anualmente ou sempre que se verifiquem alterações significativas nos procedimentos. As propostas de alteração devem ser submetidas à Direção para aprovação.

## 3. Procedimentos de Candidatura

### 3.1 Receção de Candidaturas

**Responsável:** Técnico Administrativo

**Fluxo do Processo:**

| Etapa | Ação | Prazo |
|-------|------|-------|
| 1 | Receção do formulário de candidatura (presencial, correio ou online) | D+0 |
| 2 | Verificação da documentação anexa | D+2 |
| 3 | Registo no sistema de gestão de candidaturas | D+2 |
| 4 | Envio de confirmação de receção ao candidato | D+3 |
| 5 | Se documentação incompleta: solicitação de documentos em falta | D+5 |
| 6 | Encaminhamento para análise técnica | D+7 |

**Documentos necessários:**

| Documento | Obrigatório | Observações |
|-----------|-------------|-------------|
| Formulário de candidatura preenchido | Sim | Assinado |
| Cópia do documento de identificação | Sim | Válido |
| Comprovativo de morada | Sim | Últimos 3 meses |
| Declaração de IRS ou comprovativo de rendimentos | Sim | Último ano |
| Portfolio ou fotografias de trabalhos | Sim | Mínimo 5 exemplos |
| Certificados de formação | Se aplicável | |
| Cartas de recomendação | Opcional | |

### 3.2 Verificação de Elegibilidade

**Responsável:** Técnico de Serviço Social

**Critérios a verificar:**

| Critério | Verificação | Fonte |
|----------|-------------|-------|
| Idade ≥ 18 anos | Data de nascimento | Documento de identificação |
| Ofício tradicional | Portfolio, certificados | Documentação anexa |
| Carência habitacional | Situação atual | Formulário, visita |
| Rendimentos | Limite máximo | IRS, declarações |
| Não proprietário | Ausência de imóveis | Declaração, consulta AT |
| Não beneficiário de outro programa | Últimos 5 anos | Declaração, consulta SS |

**Resultado:**
- Elegível → Encaminhar para seleção
- Não elegível → Comunicar ao candidato com fundamentação

| Membro | Função | Nomeação |
|--------|--------|----------|

**Responsável:** Comissão de Seleção

**Fluxo do Processo:**

| Etapa | Ação | Prazo |

| 2 | Análise individual de cada candidatura | D+14 |
| 3 | Reunião da Comissão para discussão | D+21 |
| 4 | Convocação de candidatos para entrevista | D+28 |
| 5 | Realização das entrevistas | D+35 |
| 6 | Visitas aos locais de trabalho (se aplicável) | D+42 |
| 7 | Reunião final da Comissão | D+49 |
| 8 | Deliberação e elaboração de ata | D+49 |
| 9 | Comunicação aos candidatos | D+56 |

### 4.3 Grelha de Avaliação

| Critério | Peso | Pontuação (0-100) | Ponderado |
|----------|------|-------------------|-----------|
| Domínio técnico do ofício | 30% | | |
| Situação socioeconómica | 25% | | |
| Compromisso com transmissão de saberes | 20% | | |
| Projeto de atividade | 15% | | |
| Ligação à comunidade | 10% | | |
| **TOTAL** | **100%** | | |

### 4.4 Entrevista

**Guião de Entrevista:**

| Tema | Questões-tipo | Tempo |
|------|---------------|-------|
| Percurso profissional | Como aprendeu o ofício? Há quantos anos o exerce? | 10 min |
| Situação atual | Onde trabalha atualmente? Quais as principais dificuldades? | 10 min |
| Projeto de atividade | O que pretende desenvolver no Programa? | 10 min |
| Transmissão de saberes | Está disponível para ensinar? Tem experiência? | 10 min |
| Motivação | Por que se candidata ao Programa? | 5 min |
| Questões do candidato | Dúvidas sobre o Programa | 5 min |

## 7. Procedimentos de Avaliação

### 7.1 Avaliação Anual dos Beneficiários

**Responsável:** Técnico de Serviço Social

**Período:** Setembro-Outubro de cada ano

**Procedimento:**

| Etapa | Ação | Prazo |
|-------|------|-------|
| 1 | Recolha de dados de acompanhamento | Set |
| 2 | Verificação de cumprimento de obrigações | Set |
| 3 | Preenchimento da grelha de avaliação | Set |
| 4 | Reunião de avaliação com o beneficiário | Out |
| 5 | Comunicação do resultado | Out |
| 6 | Registo no processo individual | Out |

**Grelha de Avaliação Anual:**

| Critério | Peso | Indicadores | Pontuação |
|----------|------|-------------|-----------|
| Exercício ativo do ofício | 30% | Horas de trabalho, produção, vendas | 0-100 |
| Cumprimento dos horários | 20% | Abertura ao público, encerramentos | 0-100 |
| Transmissão de saberes | 25% | Aprendizes, workshops, demonstrações | 0-100 |
| Participação em eventos | 15% | Presenças, colaboração | 0-100 |
| Conservação dos espaços | 10% | Estado de conservação, limpeza | 0-100 |

**Classificação:**

| Pontuação | Classificação | Consequência |
|-----------|---------------|--------------|
| 80-100 | Excelente | Renovação automática |
| 60-79 | Bom | Renovação |
| 40-59 | Satisfatório | Renovação com plano de melhoria |
| 20-39 | Insuficiente | Aviso, plano de recuperação |
| 0-19 | Mau | Não renovação / Resolução |

### 7.2 Avaliação do Programa

**Responsável:** Coordenador do Programa

**Frequência:** Anual

**Indicadores:**

| Indicador | Meta | Resultado |
|-----------|------|-----------|
| Taxa de ocupação | ≥ 90% | |
| Taxa de cobrança de rendas | ≥ 95% | |
| Satisfação dos beneficiários | ≥ 80% | |
| Aprendizes formados | ≥ 1 por beneficiário | |
| Eventos realizados | ≥ 6 por ano | |
| Visitantes aos espaços | ≥ 500 por ano | |

| Etapa | Ação | Prazo |
